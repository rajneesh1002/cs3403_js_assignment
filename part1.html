Internal JavaScript

  • In Internal JavaScript, JavaScript code is placed in the head and body section of an HTML page.

    External JavaScript

  • In External JavaScript, JavaScript code are stored in separate external file using the .js extension (Ex: external.js).
  • External scripts are practical when the same code is used in many different web pages.
  • To use an external script, put the name of the script file in the src (source) attribute of a script tag.
